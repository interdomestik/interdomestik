## Weekly Pilot KPI Sheet

| KPI                               | Definition                                                            | Owner            | Frequency    | Target                       | Data Source                                                                    |
| --------------------------------- | --------------------------------------------------------------------- | ---------------- | ------------ | ---------------------------- | ------------------------------------------------------------------------------ |
| Claim Start Rate                  | % of pilot members creating at least one claim                        | Admin            | Weekly       | ≥ 30%                        | Member dashboard/read-model                                                    |
| Claims Submitted                  | Count of new claims in pilot branch                                   | Admin            | Daily/Weekly | Within planned cohort volume | Member claims list/read-model                                                  |
| Agent First Response Time         | Median time from claim submission to first agent action               | Agent Lead       | Daily/Weekly | ≤ 2 operating hours          | Agent activity timestamps when available; otherwise manual pilot log sheet     |
| Staff Triage Time                 | Median time from claim submission to first staff triage               | Staff Lead       | Daily/Weekly | ≤ 4 operating hours          | Staff claims queue timestamps                                                  |
| 1 Operating-Day Update Compliance | % triaged claims with member-visible update within 24 operating hours | Staff Lead       | Daily/Weekly | ≥ 90%                        | Staff queue + member timeline                                                  |
| 2 Operating-Day Progression Rate  | % claims moved beyond initial state within 48 operating hours         | Staff Lead       | Weekly       | ≥ 85%                        | Staff status progression read-model                                            |
| Reopen Rate                       | % claims reopened after progress/resolution                           | Staff Lead       | Weekly       | ≤ 10%                        | Staff claims history                                                           |
| Handoff Completeness              | % agent handoffs containing required context                          | Agent Lead       | Weekly       | ≥ 95%                        | Agent handoff record fields when available; otherwise manual handoff checklist |
| Readiness Marker Integrity        | % key pages exposing expected readiness markers                       | QA Owner         | Daily        | 100%                         | Playwright E2E gate output                                                     |
| E2E Gate Pass Rate                | % `pnpm e2e:gate` runs passing                                        | QA Owner         | Daily/Weekly | 100%                         | CI and local gate logs                                                         |
| Security Guard Pass Rate          | % `pnpm security:guard` runs passing                                  | Engineering Lead | Daily/Weekly | 100%                         | CI and local security logs                                                     |
| Critical Data/Security Incidents  | Count of Sev1 incidents (privacy/tenant/data integrity)               | Admin            | Daily/Weekly | 0                            | Incident register + guardrail outputs                                          |

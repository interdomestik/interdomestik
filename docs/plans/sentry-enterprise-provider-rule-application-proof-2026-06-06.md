## Method

The repo added audited node CLI support in `scripts/sentry-enterprise-alerts.mjs`. `apply` requires
`--reuse-d07-routing`, reads the existing remote D07 alert owner and warning/critical actions,
normalizes only Sentry action payload fields, and writes sanitized output that does not include
action target identifiers, recipients, secrets, raw PII, claim content, or document content.

Commands:

```bash
node scripts/run-with-dotenv.mjs node scripts/sentry-enterprise-alerts.mjs apply --reuse-d07-routing --json
node scripts/run-with-dotenv.mjs node scripts/sentry-enterprise-alerts.mjs check --json
```
